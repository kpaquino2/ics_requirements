## How to run

1. Clone the repository

```bash
git clone https://github.com/kpaquino2/icy/
```

2. Go to code directory

```bash
cd code
```

3. Install the dependencies.

```bash
pnpm install
```

3. Set up your .env file

```bash
cp .env-example .env
# now edit the .env file
```

4. Run in development mode

```bash
pnpm dev
```
